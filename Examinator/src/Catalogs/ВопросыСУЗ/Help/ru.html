Создавать вопросы можно в форме «Вопросы для тестирования» или в форме теста при помощи кнопки «Создать».

При добавлении нового вопроса открывается форма создания вопроса. Создание вопроса начинается с указания его названия и выбора типа вопроса. По умолчанию выбор типа вопроса производится из следующих:

  • Один из многих. Позволяет выбрать один верный вариант ответа из нескольких предложенных.
  • Многие из многих. Позволяет выбрать несколько верных ответов из нескольких предложенных.
  • Последовательность. Позволяет построить ответ в виде логически обоснованной последовательности элементов.
  • Работа в программе. Пользователь выполняет предписанные действия в учебной базе изучаемой им конфигурации на платформе 1С:Предприятие 8, а результат этих действий автоматически проверяется.

Дополнительные типы вопросов: По шаблону, Таблица, Соответствие, Открытый, Элемент ActiveX, Психологический.

Для того чтобы дополнительные типы вопросов стали доступны, требуется установить флажок «Показывать дополнительные типы вопросов» на закладке «Дополнительно» формы вопроса.

1. В форме нового вопроса на закладке «Текст» заполняются текстовой информацией с возможностью включения картинок и символов следующие поля:

  • «Задание». Текст вопроса. Рекомендуется формулировать его таким образом, чтобы при подстановке вариантов ответов в конец высказывания фраза в целом давала истинное или ложное высказывание.
  • «Комментарий к ответу». Текст, который появляется после ответа пользователя на вопрос. В учебных вопросах здесь можно ввести правильный ответ, чтобы пользователь мог сравнить свой ответ с правильным. Также в это поле можно ввести рекомендации и советы пользователю.
  • «Подсказка». Текст подсказки, которую пользователь при желании может открыть. За использование подсказки можно установить штраф.
  • «Описание». Произвольный текст, который недоступен учащемуся. В это поле преподаватель может внести методическую информацию или служебные комментарии.
  • «Файл или набор файлов». Картинка, видеоролик, файл любого типа или набор файлов, которые будут показаны перед текстом вопроса.

2. Закладка «Варианты ответов». Варианты ответов добавляются при помощи кнопки «Создать».

2.1. Закладка «Вариант» формы «Вариант ответа на вопрос».

  • Флажок «Верный» устанавливается для верного варианта ответа. В вопросах типа «Последовательность» данный флажок не активен.
  • Текстовое поле. В нем указывается текст варианта ответа. Хорошей практикой являются относительно длинные вопросы и относительно короткие варианты ответа к ним. Текст варианта ответа должен продолжать текст вопроса или соответствовать ему.
  • Поле «Файл или набор файлов» - картинка или другой объект, который будет показан в качестве варианта ответа.

2.2. Закладка «Комментарии» формы «Вариант ответа на вопрос». Для каждого варианта ответа (кроме вопросов типа «Последовательность») можно указать верный и ошибочный комментарий. Эта возможность используется для указания типичных ошибок пользователя, приводящих к выбору им неправильного ответа, или для пояснения причин правильности верного варианта ответа. Если несколько вариантов ответа имеют комментарии для отображения, то после ответа на вопрос будут отображены все комментарии один под другим.

  • Поле «Верный комментарий». Текст и/или картинка (или другой ресурс), которые будут показаны под вариантами ответа, если это:
    • верный вариант ответа и пользователь выбрал его;
    • неверный вариант ответа и пользователь его НЕ выбрал.
  • Поле «Ошибочный комментарий». Текст и/или картинка (или другой ресурс), которые будут показаны под вариантами ответа, если это:
    • неверный вариант ответа и пользователь выбрал его;
    • верный вариант ответа и пользователь его НЕ выбрал.

Кнопка «Создать группу» закладки «Варианты ответов».

Для создания группы (или, иначе, папки) следует на закладке «Варианты ответов» нажать кнопку «Создать группу». В открывшейся форме «Варианты ответов на вопрос» необходимо указать название создаваемой группы и значение параметра «Количество доступных вариантов». Варианты ответов могут быть перенесены в группу с помощью перетаскивания мышью.

Когда варианты ответов объединены в группы и параметром «Количество доступных вариантов» задано количество вариантов ответов, то при формировании вопроса из каждой группы выбирается указанное количество вариантов ответа. Эта возможность используется для создания вопросов с изменяющимися при каждом показе вариантами ответов. Например, можно 10 вариантов ответов объединить в две группы по пять вариантов и указать, что из каждой группы требуется выбирать по два варианта ответа. Тогда в сформированном вопросе пользователь увидит четыре варианта ответа, случайно выбранные из двух групп.

3. Закладка «Настройки».

Установка флажка «Перемешивать варианты» позволяет при каждом новом показе вопроса выводить варианты ответа в случайном порядке.

  • При установке флажка «Разрешен развернутый ответ» пользователю кроме вопроса и вариантов ответа будет показано поле для ввода развернутого текстового ответа. Этот текстовый ответ должен быть вручную проверен преподавателем. При проверке преподаватель имеет возможность изменить автоматически рассчитанную оценку за тест. Отметим, что флажок «Исключить открытые вопросы» на закладке «Настройки» формы «Тест» имеет более высокий приоритет, «сильнее», чем флажок «Разрешен развернутый ответ». Если в тест, имеющий настройку «Исключить открытые вопросы», включены вопросы с настройкой «Разрешен развернутый ответ», то поле для ввода текстового ответа показываться НЕ будет.
  • Поле «Метод расчета результата», доступное только для вопросов типа «Многие из многих» и «Последовательность», допускает выбор следующих методов расчета итогового балла за ответ на вопрос: «Процент верных» или «Точное совпадение». При выборе «Процент верных» за ответ на вопрос типа «Многие из многих», при ответе на который не выбран ни один неправильный ответ, будет выставлено количество баллов, соответствующее проценту выбранных правильных ответов от общего числа верных ответов в этом вопросе (если выбран хотя бы один неправильный ответ, то за вопрос начисляется 0 баллов). При выборе «Процент верных» за ответ на вопрос типа «Последовательность» будет выставлено количество баллов, соответствующее проценту правильно установленных в последовательности вариантов ответов от общего числа вариантов ответов. По методу «Точное совпадение» за ответ на вопрос дается 100 баллов при полностью верном ответе и 0 баллов в случае одной или более ошибок.
  • Параметр «Штраф за подсказку» устанавливает количество баллов, которое будет вычтено из итогового балла ответа на вопрос, если пользователь воспользуется подсказкой. Параметр может иметь значение от 0 до 100 баллов.

4. Закладка «Вид».

Настройки на этой закладке влияют на представление вопроса:

  • Установка флажка «Отображать наименование» позволяет вывести над текстом вопроса его наименование.
  • Если установлен флажок «Горизонтальное размещение вариантов», то варианты ответов располагаются по горизонтали. В противном случае варианты ответов располагаются по вертикали.
  • С помощью установки флажка «Отображать имена разделов вопроса» вопрос приобретает более структурированный, табличный вид. В левой части экрана будут выведены названия разделов вопроса (Задание, Ответ, Результат, Комментарий и др.).
  • Поле «Текст раздела с вариантами ответа» позволяет предварить перечисление вариантов ответа некоторым текстом. Например, можно написать «Укажите один правильный вариант ответа» или «Укажите все правильные варианты ответа».
