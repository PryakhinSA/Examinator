Для каждого элемента содержания электронного ресурса задается один из объектов представления информации: Подборка, Страница, Ресурс, Файл, Фрейм, Глоссарий или Шаблон поиска. При создании группы элементов, можно указать карту, которая будет отображать структуру группы с необходимым уровнем детализации.
