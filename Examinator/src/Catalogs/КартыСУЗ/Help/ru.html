Карта предназначена для удобного отображения на одной странице сложных иерархических объектов, таких как «Электронный ресурс» и «Подборка». Элемент справочника «Карты» несет в себе информацию только о способе и настройках отображения карты, но не об элементе, карту которого следует построить. У тех элементов, карту которых можно построить, существует реквизит «Карта», привязка к которому карты с нужными настройками позволит сформировать ее для данного элемента. Карту можно сформировать для электронного ресурса, группы элементов содержания и подборки.

Для карты можно указать «Скрывать ресурс с одним файлом». При установке данного флажка при отображении на карте ресурса, содержащего всего один файл, данный файл не будет отображен, а ресурс будет содержать ссылку на данный файл. Следует обратить внимание, что ресурс не будет показан только в случае, если он является элементом подборки и не находится на самом верхнем уровне данной подборки.

Пользователь имеет возможность просматривать карту с минимальной или максимальной детализацией. Увеличить или уменьшить уровень детализации можно с помощью кнопки в правом верхнем углу окна карты . Элементы карты отображаются в виде ссылок, при помощи которых можно перейти к элементу. Способ открытия ссылок карты можно настроить при помощи кнопки в правом верхнем углу окна карты.
