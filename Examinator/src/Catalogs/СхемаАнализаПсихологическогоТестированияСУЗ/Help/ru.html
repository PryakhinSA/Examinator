Если тест содержит вопросы типа "Психологический", то при помощи справочника "Схема анализа психологического тестирования" можно задать правила вывода комментариев на основе анализа суммы набранных пользователем баллов за психологические вопросы.

Психологические вопросы можно комбинировать с обычными вопросами в рамках одного теста, они не будет влиять на расчет итогового результата.  Если тест содержит только психологические вопросы, то результат всегда будет равен 100%.
