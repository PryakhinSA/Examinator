При разработке курса все или часть материалов можно загружать из внешних файлов. Для этого их необходимо подготовить и разместить в одном каталоге, который дальше называем каталог «Теория».

Файлы во время загрузки могут быть преобразованы или загружены в неизменном виде в зависимости от флажка «Преобразовать». Важно отметить, что теорию, загруженную из внешних файлов, можно изменить только путем внесения правок во внешние файлы и повторной загрузки их в теорию электронного курса.

Преобразование возможно только для файлов MS Word и MS PowerPoint. Преобразование doc и docx файлов в электронный курс использует возможности MS Word в части сохранения файлов как «веб-страница с фильтром», а преобразование ppt и pptx файлов в электронный курс использует возможности MS PowerPoint в части сохранения презентации в виде набора картинок. Корректность преобразования файлов в электронный курс можно предварительно проверить средствами пакета MS Office с помощью их сохранения в описанных выше форматах.

Единичный файл MS Word преобразуется в один элемент теории с единственной страницей. Каталог файлов преобразуется в один элемент теории с числом страниц по количеству файлов в каталоге. Презентация MS PowerPoint преобразуется в один элемент теории с числом страниц по количеству слайдов презентации. На каждой странице курса будет показан один статичный слайд (анимация не поддерживается).

Файлы без установленного флажка «Преобразовать» загружаются в курс «как есть» и могут быть в дальнейшем открыты из курса в той программе, которая предназначена для открытия этого типа файлов на компьютере пользователя.

Исходные файлы и каталоги с теоретической информацией размещаются в каталоге «Теория» общим линейным списком без учета их иерархии и будущего порядка в курсе. В дальнейшем после их загрузки в курс можно настроить иерархию и порядок элементов теории, загруженных из внешних файлов, т.е. настроить по замыслу методиста навигационное дерево электронного курса. При повторной загрузке созданная в курсе структура теории не потеряется.

Если требуется в один элемент теории поместить несколько файлов, то их необходимо разместить в одном каталоге, входящем в состав каталога «Теория». Рекомендуется для каждого элемента теории даже с одной страницей создавать свой каталог исходных файлов, который будет содержать всего один файл. Тогда при необходимости добавить в элемент теории новые страницы можно будет легко добавить в этот каталог новые файлы. Исключение составляют презентации, которые почти всегда имеет смысл размещать на верхнем уровне каталога «Теория», так как при конвертации каждый слайд презентации становится самостоятельной страницей элемента курса.

В документах MS Word, которым при загрузке в курс установлен флажок «Преобразовать», можно средствами MS Word (вставка/гиперссылка) создавать ссылки на другие файлы из каталога «Теория» или на страницы сайтов. НЕЛЬЗЯ размещать гиперссылку на файл, не содержащийся в каталоге «Теория». После загрузки внешних файлов ссылки будет работоспособны только в том случае, если на форме загрузки файлов были размещены все файлы, на которые есть ссылки в теории курса. Другими словами, если теория электронного курса была загружена, например, из 100 разных внешних файлов, то при необходимости внести исправление всего на одну загруженную страницу теории, следует:

  • внести исправление в файл, из которого была загружена подлежащая правке страница теории;
  • поместить на форму загрузки все 100 файлов теории,
  • выполнить повторную загрузку теории из внешних файлов (неизмененные файлы в базу могут не загружаться, но на форме загрузки они должны быть представлены для контроля ссылок).

Если требуется вставить видеоролик на страницу теории, то необходимо средствами MS Word создать обычную гиперссылку на него. Если ссылку на видеоролик создать с текста, заключенного в двойные фигурные скобки, (например, {{Ссылка на видеоролик}}), то окно видеоплеера будет отображаться непосредственно на странице теории. Важно, что двойные фигурные скобки должны быть частью ссылки. Если двойных фигурных скобок в тексте, с которого установлена гиперссылка, нет, то такая ссылка сохранится на странице теории, и видеоролик можно будет просмотреть кликом по ней.

В документах MS Word, которым при загрузке в курс установлен флажок «Преобразовать», можно размещать различные иллюстрации. Исходный размер (ширина и высота) файлов иллюстраций, размещаемых в doc или docx файлах, должен настраиваться так, чтобы они умещались на мониторах, с которых пользователи могут изучать электронный курс. Например, если минимальное разрешение мониторов пользователей выбрано разработчиками курса как 1280х1024, то размер картинок, размещаемых в doc или docx файлах теории, не должен превышать примерно 1000х800. При размещении в doc или docx файлах картинка может быть уменьшена средствами MS Word до выбранного методистом размера. После загрузки файлов в электронный курс картинка показывается на странице теории в уменьшенном размере, но по клику на неё будет увеличиваться до своего исходного размера.

Рассмотрим настройки формы «Загрузка документов».

Файлы и папки в форму удобно добавлять путем перетаскивания их из проводника или другого файлового менеджера. Также можно воспользоваться кнопками «Добавить файлы» и «Добавить каталог».

Для файлов и папок в табличной части формы доступны следующие флажки:

  • «Загружать» . Если флажок установлен, то файл или папка со всем своим содержимым будут загружены в теорию курса. Когда теория загружается в курс впервые, то флажок «Загружать» следует установить у всех файлов и папок. Если файлы и папки загружаются в теорию курса повторно (например, после правок), то для сокращения времени загрузки флажок «Загружать» можно установить только у файлов, которые изменились. При помещении файлов и папок в таблицу флажок «Загружать» устанавливается автоматически у ранее не загружавшихся файлов и у файлов, дата сохранения которых изменилась по сравнению с датой сохранения ранее загруженного файла. Если у методиста есть сомнения в том, обновлялся отдельный файл или нет, то для этого файла следует установить флажок загрузки.
  • «Преобразовать». Флажок устанавливается, если требуется преобразовать doc или docx файл в страницу с теорией, а ppt или pptx файл - в набор страниц со слайдами. Если флажок установлен у каталога, то он будет преобразован в элемент теории, а каждый файл в каталоге - в страницу теории. В противном случае каталог будет загружен как набор файлов, открываемых внешними приложениями. Например, ppt файл будет показан в курсе ссылкой, открываемой установленным на компьютере пользователя MS PowerPoint.
  • «Скрыть». Флажок устанавливается для тех файлов, которые не должны отражаться на закладке «Содержание» курса и, соответственно, в его навигационном дереве. Обычно это файлы или страницы, на которые есть внутренние гиперссылки из теории курса, например, видеоролики или выдержки из нормативных документов. Файлы с установленным флажком «Скрыть» после загрузки в теорию будут автоматически добавлены на закладку «Зависимые ресурсы» данного электронного курса.

Установка флажка «Загружать новые» позволяет вернуться к автоматически определяемым установкам флажка «Загружать» для файлов и папок, размещенных в табличной части формы. Например, методист вручную менял установки флажков «Загружать» у файлов объемистой теории и ошибся, а потом запутался. С помощью установки флажка «Загружать новые» он может вернуться к исходным настройкам загрузки файлов.

Также на форме доступны параметры «Загрузить в…», в которых можно задать группу элементов теории (папку навигационного дерева курса), в которую будут загружены внешние файлы.

Для начала загрузки внешних файлов теории необходимо нажать кнопку «Загрузить».
