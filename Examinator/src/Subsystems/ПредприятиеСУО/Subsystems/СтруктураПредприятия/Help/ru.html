Структура предприятия

В рамках информационной системы можно формировать структуру предприятия при помощи следующих справочников:

  • "Организации";
    • "Подразделения организации"
  • "Подразделения предприятия";
  • "Должности".

Поддерживается учет структуры предприятия по центрам ответственности и по юридическим лицам. Оба способа являются независимыми, то есть можно использовать либо один, либо оба сразу.

При формировании структуры предприятия по центрам ответственности в справочнике "Подразделения предприятия" создается иерархическая структура подразделений (центров ответственности).

При формировании структуры предприятия по юридическим лицам в справочнике "Организации" создаются юридические лица, а в рамках каждого юридического лица указывается его структура.

В информационной базе по умолчанию существует основная организация и можно создавать требуемое количество дополнительных организаций.

Структура каждой организации создается аналогично структуре предприятия, при этом поддерживается вложенность подразделений.

Сотрудники

Все сотрудники обязательно должны быть пользователями информационной базы. Данное требование связано с тем, что многие действия, например прохождение электронных курсов, могут выполняться только пользователями информационной базы. Пользователь информационной базы не обязательно может быть сотрудником. Следует обратить внимание, что если пользователь не является сотрудником, то он не будет фигурировать в некоторых вариантах отчетов, которые ориентированы на анализ данных в разрезе структуры организации. Для просмотра структуры предприятия и списка сотрудников используется отчет "Сотрудники".

Список сотрудников организации доступен при помощи команды "Сотрудники" на панели навигации. Допускается заведение новых сотрудников в информационной базе вручную или при помощи инструмента "Импорт сотрудников" в разделе "Сервис". При заведении нового сотрудника в информационную базу вручную, обязательно указываются следующие данные: дата назначения, пользователь информационной базы и должность. Опционально указываются: подразделение предприятия, организация и подразделение организации. Если пользователь информационной базы, соответствующий этому сотруднику, еще не создан, то в поле "Имя пользователя" можно вписать имя и после записи сотрудника будет создан новый пользователь и установлено соответствие с созданным сотрудником.
