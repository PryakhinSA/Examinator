Контрольные вопросы являются важной частью почти любого курса, так как позволяют сделать его интерактивным, а значит более эффективным.

Контрольные вопросы типа «Один из многих», «Многие из многих» или «Последовательность» можно подготовить во внешнем файле MS Excel. В xls или xlsx файле с вопросами:

  • каждая строка таблицы соответствует одному вопросу,
  • каждый лист файла соответствует одному тесту в электронном курсе.

Первая строка в таблице должна содержать имена колонок. По умолчанию программа понимает следующие имена колонок:

  • «Код». * Уникальный код вопроса. Необходимо обеспечить уникальность кода каждого вопроса.
  • «Задание». * Формулировка вопроса.
  • «Наименование». Наименование вопроса в справочнике «Вопросы». Если поле отсутствует, то в качестве наименования будут указаны первые 150 символов из поля «Задание».
  • «Ответ». Текст комментария, который появится после любого ответа пользователя на вопрос.
  • «Подсказка». Текст подсказки.
  • «Описание». Служебный текст, который не виден обучающимся. Например, методические замечания к вопросу.
  • «ВариантОтвета». Каждый вариант ответа указывается в отдельной колонке с именем «ВариантОтвета». Вместо текста допускается указывать путь к файлу иллюстрации из ZIP архива для представления варианта ответа в виде картинки, например, «img/62-1.gif».
  • «ВерныйВариант». Указывается порядковый номер колонки (из колонок «ВариантОтвета») верного варианта ответа, например, «1» или через запятую нескольких верных вариантов, например, «1,3».
  • «ТипВопроса». Указывается «ОдинИзМногих», «МногиеИзМногих» или «Последовательность». Если тип вопроса не указан, то он определяется по количеству верных ответов. Если верных ответов больше одного, то выбирается тип «Многие из многих», в противном случае выбирается «Один из многих».
  • «ПеремешиватьВарианты». Указывается «Да» или «Нет».
  • «РазрешенРазвернутыйОтвет». Указывается «Да» или «Нет».
  • «ШтрафЗаПодсказку». Указывается в баллах от 0 до 100.
  • «МетодРасчетаРезультата». Указывается «ТочноеСовпадение» или «ПроцентВерных».
  • «Ресурс». Указывается путь к файлу (картинки или видеофайлу) из ZIP архива. Файл будет прикреплен к вопросу и отображен при его просмотре перед формулировкой вопроса.
  • «ОтображениеВариантов». Влияет на внешний вид вариантов ответов, подробнее описывается в главе 4. Принимает значение «Текст» (это настройка вида вариантов ответа по умолчанию), «Наименование», «Номер» или «Код».
  • «Пропустить». Опция позволяет отметить вопросы, которые не требуется загружать в базу. У таких вопросов нужно установить значение «Да».
  • «Рубрикатор». Наименование рубрикатора, к которому относится вопрос.
  • «ЭлементРубрикатора». Указывается тема рубрикатора, к которой относится вопрос. Допускается указание нескольких тем одного рубрикатора в смежных колонках с одинаковым именем «ЭлементРубрикатора».
  • «ВесВопроса». Вес вопроса в данном тесте или в теме рубрикатора, к которой отнесен вопрос.

Знаком «*» отмечены обязательные колонки.

Если в вопросах есть ссылки на файлы (картинки), то эти файлы необходимо запаковать в один архив формата ZIP и указать к нему путь на форме «Загрузка вопросов».

Существует два подхода при разработке вопросов во внешних файлах:

  1. Представление каждого листа файла Excel  в виде отдельного теста. В этом случае после загрузки тесты будут сформированы автоматически. Этот подход удобен для небольших курсов, где содержание каждого теста определено заранее.
  2. Создание вопросов курса на одном листе файла Excel с разбивкой их по темам рубрикатора. Затем в курсе из загруженных вопросов вручную создаются тесты, в которых указывается количество вопросов и темы рубрикатора, из которых они формируются. Этот подход подходит для больших курсов, в которых удобно сначала создать базу контрольных вопросов, а потом из них создавать тесты по отдельным темам.

Важно отметить, что в отличие от теории, загруженной в курс путем преобразования файлов, при загрузке вопросов и тестов из файлов MS Excel созданные в информационной базе вопросы и тесты МОЖНО редактировать в общем порядке. Иначе говоря, с вопросами и тестами, загруженными в курс из файлов MS Excel, методист может работать совершенно так, как с вопросами и тестами, которые были созданы непосредственно в программе.

В форме загрузки вопросов указывается путь к xls или xlsx файлу с вопросами, а также, если в вопросах или вариантах ответов есть ссылки на файлы иллюстраций, то и путь к архиву ZIP с файлами. Также на форме доступны параметры «Загрузить в…», в которых можно указать, в какую группу элементов курса будут помещены загружаемые тесты.

Закладка «Тесты» автоматически заполняется всеми тестами, которые содержатся в загружаемом файле; в каждой строке один тест. Данная закладка появится в форме загрузки вопросов только в том случае, если файл загружается в электронный курс (тесты будут добавлены в этот курс). Установкой флажка «Скрыть» можно не помещать отдельный указанный тест в содержание электронного курса и, соответственно, не отображать его в навигационном дереве курса. Поле «Тест для обновления» для повторно загружаемых обновленных тестов автоматически заполняется сведениями о том тесте, который будет обновлен. Если в силу разных причин поле «Тест для обновления» автоматически не было заполнено, то есть возможность вручную выбрать и указать в нем подлежащий обновлению тест.

Закладка «Вопросы» автоматически заполняется всеми вопросами, которые содержатся в загружаемом файле; в каждой строке один вопрос. Поле «Вопрос для обновления» для повторно загружаемых обновленных вопросов автоматически заполняется сведениями о том вопросе, который надлежит обновить. Если в силу разных причин поле «Вопрос для обновления» автоматически не было заполнено, то есть возможность вручную выбрать и указать в нем подлежащий обновлению вопрос.

На закладке «Настройки» доступны следующие настройки.

Если установлен флажок «Сформировать тесты», то по каждому листу файла Excel создается тест. Если флажок снят, то выполняется только загрузка в информационную базу вопросов со всех листов файла MS Excel, а тесты не создаются.

Кроме того, на закладке «Настройки» размещена таблица с перечнем имен колонок, которые имеются в загружаемом файле с вопросами. Если имена колонок отличаются от рекомендуемых, то в этой таблице нужно установить их соответствие с рекомендуемыми именами.

Для начала загрузки вопросов и тестов необходимо выбрать команду «Загрузить».
