Для просмотра структуры предприятия используется отчет "Сотрудники".

Присутствуют следующие варианты этого отчета:

  • Список сотрудников. Общий список сотрудников с указанием организации, подразделения организации, подразделения предприятия и должности.
  • Сотрудники организаций. Список сотрудников в соответствии со структурой юридических лиц.
  • Сотрудники предприятия. Список подразделений организации с указанием сотрудников каждого из них.
