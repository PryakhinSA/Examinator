Доступны следующие настройки:

  • Изменить пароль. Позволяет установить новый пароль.
  • Сведения о пользователе. Позволяют настроить свой профиль: указать пароль и имя пользователя для входа в программу (логин), изменить контактные данные.
  • Очистить каталог временных файлов. Используется для удаления временных файлов, создаваемых программой, с целью экономии дискового пространства или в случае повреждения временных файлов.
  • Удалить персональные настройки. Удаляются все персональные настройки, которые пользователь установил для электронных ресурсов при их использовании.
