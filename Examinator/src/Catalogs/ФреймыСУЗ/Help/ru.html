Данный объект позволяет описать и каталогизировать информацию произвольной природы. Это могут быть книги, журналы, интернет-источники, законодательные акты и пр. Фреймы описываются при помощи иерархической структуры свойств объекта. Свойство фрейма представлено в виде реквизитов: наименование, описание типа значения, значение.

Наименование – имя, которое будет представлено при выводе иерархии свойств.

Описание типа значения – это ссылка на элемент плана видов характеристик фреймов. Каждый элемент плана видов характеристик содержит информацию о доступных типах: строка, число, дата, булево, ссылка на другой объект. Элемент может содержать один или несколько из перечисленных типов по выбору пользователя. Для типов дата, булево, число при необходимости можно задать правила форматирования.

Значение свойства фрейма может принимать любое в рамках выбранного описания типов значений. Например, если описание типов включает тип «строка» и «дата» для свойства «Год издания», можно выбрать любой тип и указать дату либо на календаре, либо вписать в виде строкового значения.

Для разных версий фрейма можно задавать текстовое описание. Все остальные реквизиты относятся к структуре объекта и являются общими для всех версий.

Если фрейм был импортирован, то существует возможность добавлять новые характеристики без потери возможности обновления от поставщика. При обновлении фрейма от поставщика добавленные новые характеристики сохранятся.

Фрейм отображается в виде таблицы, в которой слева записываются наименование характеристики, а справа - значение свойства. Характеристики могут быть объединены в группы, тогда справа от наименования группы отображается таблица ее характеристик. Вложенность групп не ограничивается.
