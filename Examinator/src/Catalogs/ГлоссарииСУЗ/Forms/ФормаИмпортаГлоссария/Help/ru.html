Глоссарий позволяет сформировать список терминов и их определений. Для формирования глоссария во внешнем файле используется таблица MS Excel. Каждая строка таблицы соответствует одному термину, а каждый лист - одному глоссарию.

Первая строка в таблице должна содержать имена колонок. По умолчанию программа понимает следующие имена:

  • «Термин». * Указывается наименование термина. При повторной загрузке поиск терминов будет выполняться по этому полю.
  • «Определение». * Текст определения термина.
  • «Синоним». Альтернативное наименование термина. Допускается создавать несколько колонок «Синоним».
  • «Пропустить». Параметр используется для отметки терминов, которые не требуется загружать в курс. Для таких терминов в колонке указывается значение «Да».

Знаком «*» отмечены обязательные к заполнению колонки.

Важно отметить, что в отличие от теории, загруженной в курс путем преобразования файлов, при загрузке терминов и глоссариев из файлов MS Excel созданные в информационной базе термины и глоссарии МОЖНО редактировать в общем порядке. Иначе говоря, с терминами и глоссариями, загруженными в курс из файлов MS Excel, методист может работать совершенно так, как с терминами и глоссариями, которые были созданы непосредственно в программе.

В форме загрузки глоссария указывается путь к xls или xlsx файлу с вопросами и глоссариями. Также на форме доступны параметры «Загрузить в…», в которых можно указать, в какую группу элементов курса будут помещены загружаемые глоссарии.

Закладка «Глоссарии» автоматически заполняется всеми глоссариями, которые содержатся в загружаемом файле; в каждой строке один глоссарий. Установкой флажка «Скрыть» можно не помещать отдельный указанный глоссарий в содержание электронного курса и, соответственно, не отображать его в навигационном дереве курса (доступно при загрузке глоссария в электронный курс). Поле «Глоссарий для обновления» для повторно загружаемых обновленных глоссариев автоматически заполняется сведениями о том глоссарии, который надлежит обновить. Если в силу разных причин поле «Глоссарий для обновления» автоматически не было заполнено, то есть возможность вручную выбрать и указать в нем подлежащий обновлению глоссарий.

Закладка «Термины» автоматически заполняется всеми терминами, которые содержатся в загружаемом файле; в каждой строке один термин. Поле «Термин для обновления» для повторно загружаемых обновленных терминов автоматически заполняется сведениями о том термине, который надлежит обновить. Если в силу разных причин поле «Термин для обновления» автоматически не было заполнено, то есть возможность вручную выбрать и указать в нем подлежащий обновлению термин.

На закладке «Настройки» доступны следующие настройки.

  • Если установлен флажок «Сформировать глоссарии», то по каждому листу файла Excel создается глоссарий. Если флажок снят, то выполняется только загрузка в информационную базу терминов со всех листов файла MS Excel, а глоссарии не создаются.
  • Кроме того, на закладке «Настройки» размещена таблица с перечнем имен колонок, которые имеются в загружаемом файле с терминами. Если имена колонок отличаются от рекомендуемых, то в этой таблице нужно установить их соответствие с рекомендуемыми именами.

Для начала загрузки глоссария и терминов необходимо выбрать команду «Загрузить».
