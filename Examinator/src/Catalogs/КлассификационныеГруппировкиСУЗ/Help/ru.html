При последовательном представлении элементов подборки с установленной опцией "Показывать классификатор" будут отображаться только те классификаторы, которые входят в классификационную группировку. При представлении элементов в виде классификационной таблицы в столбцах таблицы будут указаны только классы из классификационной группировки. Если группа не содержит элементов какого-либо класса, который есть в классификационной группировке, то данный класс все равно будет указан в таблице.
