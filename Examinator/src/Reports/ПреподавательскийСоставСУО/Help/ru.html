Обобщенный список преподавателей можно вывести при помощи отчета "Преподавательский состав". Отчет имеет следующие варианты:

  • Количество преподавателей;
  • Общий список преподавателей;
  • Список мероприятий и преподавателей.

Вариант отчета "Количество преподавателей" позволяет в табличной форме оценить количество преподавателей каждой роли, закрепленных за мероприятиями.

Вариант "Общий список преподавателей" показывает линейный список всех преподавателей. Вариант может быть использован для удобного вывода информации о преподавательском составе на печать или экспорта в электронные таблицы.

Вариант "Список мероприятий и преподавателей" позволяет вывести список преподавателей в разрезе мероприятий. Таким образом, можно проанализировать, какие преподаватели закреплены за каждым мероприятием.
