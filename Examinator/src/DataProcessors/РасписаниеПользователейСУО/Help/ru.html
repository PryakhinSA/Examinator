Для контроля учебной нагрузки пользователей используется обработка "Расписание пользователей", которая доступна на панели "Сервис".

Информация о расписании пользователей доступна в виде диаграммы и таблицы. При помощи поля "Период" можно задать любой интересующий нас период вывода информации. Команда "Добавить пользователей" позволяет добавить в форму необходимое количество пользователей, информацию по которым мы хотим посмотреть.

События, которые отображаются на диаграмме, могут быть трех видов: "Обучение", "Преподавание" и "Личные события". При двойном щелчке мыши на событии откроется форма этого события. Если события пересекаются, это будет выделено цветом.
